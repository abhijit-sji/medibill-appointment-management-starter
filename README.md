# Appointment Management

Full-featured appointment scheduling system with calendar views, availability management, appointment types, operatories, and slot-based booking. Supports NexHealth sync and Google Calendar integration.

Version: 1.0.0
Status: stable

## Features
- Interactive calendar
- Drag-and-drop scheduling
- Availability templates
- Appointment types
- Operatory management
- Slot-based booking
